# 🛡️ SOC vs NOC vs IR Team

| Team | Main Focus | Responsibilities |
|------|-------------|------------------|
| SOC | Security Monitoring | Detecting and responding to cyber threats |
| NOC | Network Availability | Managing network uptime and performance |
| IR Team | Incident Response | Handling and investigating security incidents |

## 🔍 SOC (Security Operations Center)
SOC teams continuously monitor logs, alerts, and suspicious activities to detect cyberattacks and security threats.

### Main Tasks
- Alert monitoring
- Log analysis
- Threat detection
- Incident escalation
- SIEM monitoring

---

## 🌐 NOC (Network Operations Center)
NOC teams focus on network infrastructure and system uptime.

### Main Tasks
- Network monitoring
- Server availability
- Device management
- Performance troubleshooting

---

## 🚨 IR Team (Incident Response Team)
IR teams investigate and contain confirmed security incidents.

### Main Tasks
- Malware analysis
- Digital forensics
- Threat containment
- Recovery operations
- Root cause analysis

# ⚔️ Attack Use Cases

## Example Use Cases

### Brute Force Attack
Multiple failed login attempts from a single IP address.

### Phishing Attack
Malicious email attempting credential theft.

### Malware Infection
Suspicious executable detected on endpoint.

### Privilege Escalation
User gaining unauthorized admin access.

### Data Exfiltration
Sensitive data transferred outside organization.

# 🚨 Real-world Attack Scenario

## Scenario: Suspicious Login Activity

### Situation
A SIEM alert detects multiple failed login attempts followed by a successful login from an unusual location.

### SOC Actions
1. Analyze logs
2. Verify source IP
3. Check user activity
4. Investigate possible compromise
5. Escalate if malicious

### Possible Indicators
- Multiple failed logins
- Impossible travel login
- Unusual working hours
- Unknown device access

# ✅ True Positive vs ❌ False Positive

| Type | Meaning |
|------|----------|
| True Positive | Real security threat detected correctly |
| False Positive | Benign activity mistakenly flagged as malicious |

---

## ✅ True Positive Indicators
- Malware hash detected
- Unauthorized login
- Known malicious IP
- Data exfiltration activity
- Ransomware behavior

---

## ❌ False Positive Indicators
- Employee VPN login
- Authorized scanning activity
- Security testing
- Internal automated scripts
- Misconfigured alerts
